# 🔥 **1. Core Navigation Commands (SUPERCHARGED)**

These go beyond “open tab” — they bring *superpowers*.

### ✅ `!jump <keyword>`

Instantly jumps to an existing tab by fuzzy matching.
Example:
`!jump jira` → switches to your Jira tab instantly.

### ✅ `!go <app>` (App shortcuts)

Open popular sites with shorthand:

* `!go yt` → YouTube
* `!go gm` → Gmail
* `!go gh` → GitHub
* `!go fig` → Figma

### ✅ `!back`

Go to previous page.

### ✅ `!history <keyword>`

Search your history without leaving the tab.

### ✅ `!recent`

Shows recent tabs in a dropdown.

### ⭐ **Signature Feature: “Universal Spotlight”**

### `!spot <anything>`

Search across:

* tabs
* bookmarks
* history
* CoolDesk items
* workspace items
* team links
* commands

This will feel like **macOS Spotlight but inside Chrome**.
