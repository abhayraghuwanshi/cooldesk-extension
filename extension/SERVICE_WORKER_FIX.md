# Service Worker Registration Fix

## Issue
Service worker registration was failing with **Status code: 3** ("An unknown error occurred when fetching the script").

## Root Cause
The extension needed to be rebuilt. The `@crxjs/vite-plugin` automatically transforms the manifest during build:
- **Source manifest**: Points to `src/background/background.js`
- **Built manifest**: Points to `service-worker-loader.js` (generated by the plugin)

## Solution Applied
1. **Cleaned the dist folder**: `Remove-Item -Recurse -Force dist`
2. **Rebuilt the extension**: `npm run build`
3. **Build completed successfully** with all files generated correctly

## Files Generated
- ✅ `dist/service-worker-loader.js` (45 bytes)
- ✅ `dist/assets/background.js-DG5271zD.js` (82.38 KB)
- ✅ `dist/manifest.json` (correctly pointing to `service-worker-loader.js`)
